How It Works
Step 1: Run Network Scanner

Run the network scanner script on your network:

python3 scripts/network_scanner.py 192.168.1.0/24

This creates a network_scan.json file with discovered devices.

Step 2: Upload Scan Results

Upload the JSON file below to preview what will be imported.

Step 3: Review & Confirm

Review discovered devices and choose what to import. The system will:

  • Create new assets for devices not in your inventory
  • Update existing assets matched by MAC or IP address
  • Flag potential duplicates for your review
