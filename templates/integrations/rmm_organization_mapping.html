How RMM organization import works:
  • RMM organizations are matched by name - not by external mapping like PSA
  • During sync, if an organization with the same name exists, it will be updated
  • If no match is found, a new organization will be created
  • To prevent duplicates, use the Organization Merge Tool after syncing
  • Or manually rename organizations to match before syncing
